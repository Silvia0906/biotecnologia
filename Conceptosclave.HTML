Conceptos Clave

Lista ordenada

  1. Mejoramiento genético: Modificación del material genético de las plantas para obtener características deseables, como mayor resistencia a enfermedades, tolerancia a condiciones ambientales adversas, o mayor rendimiento.
  2. Microorganismos beneficiosos: Uso de microorganismos (bacterias, hongos, etc.) para mejorar la salud del suelo, la absorción de nutrientes por las plantas, y la protección contra patógenos.
  3. Producción de biofertilizantes: Desarrollo de fertilizantes naturales, a partir de microorganismos o de procesos biológicos, que puedan reemplazar o complementar los fertilizantes químicos.
  4. Alimentos genéticamente modificados (OGM): Plantas cuyo material genético ha sido modificado para obtener características específicas (por ejemplo, resistencia a herbicidas).
  5. Biopesticidas: Uso de sustancias naturales, como extractos vegetales o microorganismos, para controlar plagas y enfermedades.

Lista desordenada

  • Fermentación industrial: Proceso biotecnológico donde microorganismos convierten materias primas en productos útiles, como alcoholes, ácidos orgánicos o antibióticos.
  • Clonación vegetal: Técnica para reproducir plantas genéticamente idénticas a partir de células o tejidos, garantizando características uniformes y mejoradas.
  • Fitorremediación: Uso de plantas para limpiar o degradar contaminantes del suelo o agua, ayudando a restaurar ambientes dañados.
  • Bioreactores: Sistemas controlados donde se cultivan microorganismos o células para producir compuestos bioquímicos a gran escala.
  • Ingeniería metabólica: Modificación de rutas metabólicas en microorganismos o plantas para aumentar la producción de sustancias de interés, como biocombustibles o fármacos.
